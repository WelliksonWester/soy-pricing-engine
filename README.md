# ValorAgro – Calculadora de Preço da Soja

Uma solução avançada para precificação de soja, automatizando cálculos fiscais complexos e garantindo conformidade para o agronegócio.

## 🚀 Documentação Completa
Para detalhes técnicos, regras de negócio fiscais e especificações de engenharia, acesse a **Bússola do Projeto**:
👉 [docs/projeto_valoragro.md](./docs/projeto_valoragro.md)

## 🛠️ Stack Tecnológica
- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Interface:** [Tailwind CSS](https://tailwindcss.com/) & [ShadCN UI](https://ui.shadcn.com/)
- **Backend:** [Firebase](https://firebase.google.com/) (Auth & Firestore)
- **IA:** [Genkit](https://github.com/firebase/genkit)
- **Mobile:** Configurado como **PWA** (Progressive Web App)

---
*Este projeto foi desenvolvido como uma solução de engenharia para o agronegócio brasileiro.*
