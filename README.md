# ValorAgro – Calculadora de Preço da Soja

Uma solução avançada para precificação de soja, automatizando cálculos fiscais complexos e garantindo conformidade para o agronegócio.

## 🚀 Stack Tecnológica

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/) & [ShadCN UI](https://ui.shadcn.com/)
- **Backend/SaaS:** [Firebase](https://firebase.google.com/) (Auth & Firestore)
- **IA:** [Genkit](https://github.com/firebase/genkit) com Google Gemini
- **Documentação:** [jsPDF](https://github.com/parallax/jsPDF) para exportação de PDFs
- **Mobile:** Configurado como **PWA** (Progressive Web App)

## 📋 Funcionalidades

- **Calculadora Automática:** Processamento de preços baseados em farelo e óleo.
- **Gestão Fiscal:** Definição automática de CFOP, CST e alíquotas de ICMS/Funrural.
- **Exportação:** Geração de Instrução de Faturamento e Modelos de NF em PDF.
- **Configurações:** Parametrização de taxas e custos por usuário.
- **PM Canvas:** Visualização estratégica do projeto integrada ao sistema.

## 🛠️ Como rodar o projeto

1. Clone o repositório.
2. Instale as dependências com `npm install`.
3. Configure as variáveis de ambiente do Firebase em `.env`.
4. Execute o servidor de desenvolvimento com `npm run dev`.
