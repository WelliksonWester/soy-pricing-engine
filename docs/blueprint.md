# **App Name**: ValorAgro Calculator

## Core Features:

- Login Authentication: Secure login page with admin credentials (user: admin, password: admin) to access the ERP system.
- Operation Data Input: Input panel for operation details including operation type (Intraestadual, Interestadual), seller type (Produtor Rural, Comerciante), with CFOP and CST auto-filled based on selections.
- Commercial Conditions Input: Input panel for commercial conditions such as base price, industry cost, freight type (CIF/FOB), freight cost (if CIF), classification, margin, and commission.
- Tax and Calculation Simulation: Calculates gross price per bag, Funrural tax, net price per bag, and net amount to pay per bag/ton based on user inputs and selected options (Funrural Folha/Faturamento).
- Results Display: Displays calculated results including gross price per bag, freight per bag (if CIF), taxes per bag, net per bag, net per ton, and a summary of the calculation.
- Data Export: Exports the calculated data to an Excel file with columns: Operation Type, CFOP, CST, Base Price, Freight, Tax, Net/bag, Net/ton.
- CFOP Generation: Generates a pop-up displaying the CFOP code and corresponding CST based on the selected operation and vendor types.

## Style Guidelines:

- Primary color: Blue-gray (#2A5D8F) for a professional ERP feel.
- Background color: Light gray (#F3F3F3) for a clean, neutral backdrop.
- Accent color: Light gray (#D9D9D9) to provide contrast while maintaining a professional style.
- Body and headline font: 'PT Sans' (sans-serif) for clear and readable text in a technical agricultural context.
- Use a panel-based layout with rectangular blocks, consistent spacing, and rounded borders, similar to ERP interfaces.
- All fields should be vertically centered within their respective panels.
